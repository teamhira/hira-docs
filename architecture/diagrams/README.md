# Architecture Diagrams

This folder contains architectural diagrams and their source files.

## Recommended Content
- **Context Diagram**: High-level system interaction.
- **Container Diagram**: Interaction between services (SwiftUI, GraphQL, Core, Auth).
- **Sequence Diagram**: Critical flows (e.g., Auth, Quran Sync).

Always store both the diagram source (e.g., Mermaid, PlantUML, Excalidraw) and the exported image with consistent naming.
