# Initial Onboarding Checklist

## 1. Access & Setup
- [ ] Join the organization on GitHub/GitLab.
- [ ] Connect to the internal communication channel (e.g., Slack/Discord).
- [ ] Setup your local development environment according to platform guidelines.

## 2. Reading
- [ ] Read the [WHITEPAPER.md](WHITEPAPER.md).
- [ ] Review the [Architecture Documentation](architecture/README.md).
- [ ] Familiarize yourself with the [API Conventions](api/conventions.md).

## 3. First Task
- [ ] Pick a "Good First Issue" from the relevant repository.
- [ ] Submit a small PR to familiarize yourself with the CI/CD pipeline and review process.
